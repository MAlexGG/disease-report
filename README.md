# :microscope: Disease Report
A modern interactive dashboard built with React that visualizes global COVID-19 data using maps, charts, and dynamic statistics. The project focuses on data visualization, accessibility, and responsive design.

## Features
- Interactive world map with country-level data
- Color-coded severity visualization (cases per country)
- Global evolution charts (cases, deaths, recoveries)
- Country detail modal with live statistics
- Real-time data from public API
- Fully responsive design (mobile & desktop)
- Accessibility-focused visual patterns (WCAG-friendly)

## Tech Stack
- React v.19
- React Router v.7
- React Simple Maps v.3
- MUI X Charts v.9
- CSS Modules

### API Used
This project uses the free public API: https://disease.sh/docs/

It provides: Global COVID statistics, Country-specific data, Historical evolution data.

### Accessibility

The project includes accessibility improvements:

- Color gradients for data clarity
- Pattern overlays (dots, grids, stripes)
- High contrast selection states
- Visual differentiation beyond color (WCAG AA approach)

### Installation

Clone repository

``git clone https://github.com/MAlexGG/disease-report.git``

Go into project

``cd disease-report``

Install dependencies

``npm install``

Start development server

``npm run dev``
